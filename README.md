# Shuffle

청크 셔플 SMP 플러그인

---

### 게임 방식
 - /shuffle start <청크반경> <셔플주기> 명령어로 초기 설정을 진행합니다.
 - /shuffle stop 명령어로 중지가 가능합니다.
 - 플레이어가 땅 위에 있는 상태에서 셔플 발생 시 플레이어를 바뀐 청크의 맨 위쪽 좌표로 순간이동시킵니다.
